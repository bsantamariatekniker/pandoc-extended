This repository contains a Dockerfile for building an extended version of the [oficial pandoc's latex image](https://github.com/pandoc/dockerfiles).

The following is added to the build:

- Latex packages
  - titlesec
  - pagecolor
  - mdframed
  - needspace
  - sourcesanspro
  - sourcecodepro
  - ly1
  - mweights
  - titling
- Java
- Plantuml
- Python & pip
- pandoc-plantuml-filter
